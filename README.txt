Контрольная работа
По представлениям словаря
1.	Показать все представления словаря.
2.	Получить имя экземпляра.
3.	Получить имя базы данных.
4.	Получить список всех табличных пространств. 
5.	Получить список файлов табличных пространств.
6.	Получить список ролей.
7.	Получить список привилегий роли.
8.	Получить список общих пользователей.
9.	Получить список профилей безопасности.
10.	Получить список всех объектов, принадлежащих пользователю.
11.	Получить размер данных в таблице пользователя.
12.	Получить список сегментов пользователя. 
13.	Получить список выделенных экстентов для всех пользователей. 
14.	Получить количество блоков для каждой таблицы пользователя.
15.	Получить размер блока данных.
16.	Получить список PDB.
17.	Получить список параметров экземпляра.
18.	Получить список файлов журналов повтора.
19.	Получить список групп журналов повтора.
20.	Получить список архивов журналов повтора. 
21.	Получить список управляющих файлов. 
22.	Получить месторасположение файла параметров. 
23.	Получить, ведется ли архивирование файлов журналов повтора. 
24.	Получить список объектов пользовательской корзины.
25.	Получить список пользователей из файла паролей.
26.	Получить список основных областей SGA и их размер.
27.	Получить размер свободного места в разделяемом кэше.
28.	Для таблицы пользователя найти, в каком пуле буферного кэша она кэшируется. 
29.	Получить размер гранулы.
30.	Получить список сервисов.
31.	Получить список сессий.
32.	Получить список работающих в настоящее время серверных процессов.
33.	Получить список работающих в настоящее время фоновых процессов.
